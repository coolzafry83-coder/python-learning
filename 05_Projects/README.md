# Password Strength Checker 🔐

A Python-based cybersecurity project that checks password strength by analyzing different security requirements.

## Features

- Checks password length
- Detects numbers
- Detects uppercase letters
- Detects lowercase letters
- Detects special characters
- Calculates password strength score

## Requirements

- Python 3.x

## How to Run

Run the program:

```bash
python password_checker.py
